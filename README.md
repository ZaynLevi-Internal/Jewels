# Frontend - Aurielle Jewelry Website

This is the static frontend for Aurielle Jewelry.

## Deployment on GitHub Pages

1. Create a new GitHub repository.
2. Push the contents of this frontend folder to the repository.
3. Go to the repository Settings > Pages.
4. Select the main branch and root folder.
5. The site will be live at `https://yourusername.github.io/repository-name/`

## Notes

- The frontend is configured to call the backend API at `https://jewels-5zoa.onrender.com`.
- If you deploy the backend to a different URL, update the `API_BASE` in `index.html`.
